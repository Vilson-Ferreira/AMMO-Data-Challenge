# AMMO-Data-Challenge
Desafio realizado para a oportunidade da AMMO Varejo, engenharia de dados com Spark.

Aplicações:
- Processamento paralelo.
- Agregação de dados.
- Extração de dados.
- Estruturação de Spark Jobs

Ferramentas utilizadas:
- Anaconda 3.
- Python 3.6.
- Spark 2.3.1.
- Jupyter Notebook.
